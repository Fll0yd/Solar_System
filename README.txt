Solar System Project
Welcome to the Solar System project! This project provides information about the solar system, including details about the planets orbiting the sun.

Overview
The Solar System project is a simple web application created using HTML. It presents information about the planets in our solar system, their names, and their distances from the sun. The project includes an HTML file (index.html) that displays the information and an image of the solar system.

Features
Displays a brief introduction to the solar system.
Provides a table listing the planets and their distances from the sun.
Styled using CSS for a visually appealing presentation.
Includes an image of the solar system for visualization.
Usage
To view the project, simply open the index.html file in a web browser.
